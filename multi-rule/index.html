Multi-rule L-System Visualizer

Instructions

Welcome. This is a simple visualizer for multi-rule L-systems. If you haven't already checked out the basics of L-systems I encourage you to try the basic version of this app linked below. The provided settings are much the same as before, but you now have the capability of adding multiple rules. Any symbols can be used, but only the ones we have learned will be visually interpreted. If there are multiple rules with the same starting symbol only the first one will be applied. There are so many more possibilities now so take it slow and see what awesome things you can create.

  • F tells the program to draw forward Line Draw Length far
  • + and - rotate the direction the next line will be drawn by the angle
  • [ tells the program to remember it's current location.
  • ] tells the program to return to the last saved location
  • Rulebox 1 is the symbol that will be replaced
  • Rulebox 2 is what will replace the first part of the rule
  • Remove rule removes the most recently added rule
  • Scaling is meant to help you make sure the drawing stays in the canvas
  • Center will place the origin of the drawing in the center of the canvas

Note: Rules with a lot of symbols can get graphically dense quickly and cause browser lag. Be careful with how many times you hit generate, especially with the default example.
